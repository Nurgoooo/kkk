# Dodo Trainee Academy

Додо Пиццаға жаңа келген стажерлерге арналған локалды оқу сайты.

## Не бар

- Өнімдер каталогы: пицца, додстер, закуска, паста, салат, десерт, сусын.
- Әр карточкада атауы, бағасы, құрамы, қысқаша сипаттамасы және визуалы бар.
- Жаттау режимі:
  - сурет бойынша атауын табу;
  - атауы бойынша ингредиенттерін айту;
  - flashcard аудару.
- Тест статистикасы: дұрыс, қате, жалпы қаралған саны.
- Іздеу, категория сүзгісі, баға бойынша сұрыптау.
- Қазақша және орысша интерфейс.
- Жарық және қараңғы режим.
- Админ бет: `/admin`, өнімдерді JSON арқылы жаңарту.
- Прогресс пен өзгертілген өнімдер `localStorage` ішінде сақталады.

## Маңызды

Сайт сыртқы API қолданбайды. Барлық бастапқы өнімдер `data/products.json` файлында сақталған, бағалар фиксированный.

## Іске қосу

```bash
npm install
npm run dev
```

Сосын браузерден ашыңыз:

```text
http://localhost:3000
```

Админ панель:

```text
http://localhost:3000/admin
```

## Тексеру

```bash
npm run lint
npm run build
```
